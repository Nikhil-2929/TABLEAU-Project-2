This Tableau dashboard provides a comprehensive analysis of literacy rates across Tamil Nadu, categorized by cities, gender distribution, scholarship beneficiaries, and institutional resources. The visualization offers insights into female literacy rates compared to total literacy, identifying key trends and areas for improvement.

The dashboard aims to answer critical questions such as:

How does literacy vary across different cities?

What is the female literacy rate compared to the total population?

How are scholarships benefiting different regions?

How many educational institutions and teachers are available per city?

Through interactive charts, maps, and comparisons, this dashboard highlights the current state of literacy in Tamil Nadu and the factors influencing education levels across the state.
